# nvim-config

## Installation

### Remove existing configuration

For linux-based systems
```bash
# bash
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```

### Clone repository

```bash
# bash
git clone https://github.com/ctchen1999/nvim-config.git ~/.config/nvim
```
